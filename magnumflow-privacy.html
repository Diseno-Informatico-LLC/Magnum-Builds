MagnumFlow — Política de Privacidad

Aplicación móvil del sistema de gestión Magnum ERP
Última actualización: 30 de abril de 2026

1. Identidad del responsable

MagnumFlow es desarrollada y distribuida por Diseño Informático, empresa proveedora del sistema de gestión Magnum ERP.

Contacto: mauricio@gestionmagnum.com.ar

2. Naturaleza de la aplicación

MagnumFlow es una herramienta de uso interno para clientes que ya operan el sistema Magnum ERP. Se conecta exclusivamente al servidor del propio cliente (instalación on-premise o privada). No se ofrece como servicio multitenancy ni se almacenan datos en servidores controlados por nosotros.

3. Datos que la aplicación procesa

3.1. Credenciales de acceso

Para poder conectarse al servidor de su empresa, la app guarda localmente, en almacenamiento seguro del dispositivo (Keystore en Android, Keychain en iOS):

  • URL del servidor configurado por el usuario.
  • Nombre de usuario y token de sesión (JWT).

Estos datos no se transmiten a terceros y se eliminan cuando el usuario cierra sesión.

3.2. Datos de operación

La aplicación funciona con un modo offline-first: descarga catálogos (productos, clientes, monedas, etc.) y permite registrar pedidos y otros comprobantes desde el dispositivo. Esta información se almacena en una base de datos local cifrada y se sincroniza únicamente con el servidor del cliente al recuperar conexión.

3.3. Imágenes capturadas (cámara)

MagnumFlow utiliza la cámara del dispositivo, con consentimiento explícito del usuario al momento de la captura, para escanear comprobantes de compra (facturas, tickets) y adjuntarlos a un movimiento del sistema. Las imágenes:

  • Se procesan localmente en el dispositivo.
  • Se transmiten exclusivamente al servidor Magnum del cliente al confirmar el alta.
  • No se envían a servicios de terceros, ni a Diseño Informático, ni se utilizan para analítica, publicidad o entrenamiento de modelos.

4. Permisos solicitados

  • Cámara (android.permission.CAMERA): para escanear y adjuntar comprobantes. Solicitado en runtime, sólo cuando el usuario abre la pantalla de captura.
  • Internet y estado de red: para sincronizar datos contra el servidor del cliente y detectar conexión disponible.

La aplicación no solicita acceso a contactos, ubicación, micrófono, calendario, llamadas, almacenamiento externo ni a otras fuentes de datos personales.

5. Almacenamiento y transmisión

  • Los datos sensibles (credenciales, token) se cifran con el almacenamiento seguro del sistema operativo.
  • La sincronización se realiza por HTTPS contra el servidor configurado por el cliente. En redes locales privadas el cliente puede optar por HTTP plano bajo su responsabilidad.
  • No utilizamos servicios de analítica de terceros (Google Analytics, Firebase Analytics, Crashlytics, Facebook SDK, etc.).
  • No publicamos publicidad ni rastreamos usuarios entre aplicaciones.

6. Compartición con terceros

MagnumFlow no comparte datos personales ni operativos con terceros. Todos los datos viajan únicamente entre la aplicación y el servidor Magnum del cliente.

7. Retención y eliminación

Los datos almacenados localmente en el dispositivo permanecen mientras la aplicación esté instalada. El usuario puede eliminarlos en cualquier momento:

  • Cerrando sesión desde la app (limpia credenciales y caché local).
  • Desinstalando la aplicación (borra todo el almacenamiento asociado).
  • Solicitando al administrador del sistema Magnum la baja del usuario en el servidor.

8. Menores de edad

MagnumFlow es una herramienta corporativa pensada para uso de personal administrativo, comercial y de depósito. No está dirigida a menores de 13 años y no recolecta datos de menores de manera consciente.

9. Cambios en esta política

Cualquier cambio relevante se publicará en esta misma URL con la fecha de actualización modificada. Si los cambios afectan al manejo de datos personales, los notificaremos dentro de la propia aplicación.

10. Contacto

Para consultas sobre privacidad, ejercer derechos de acceso, rectificación o eliminación, o reportar incidentes, escribir a: mauricio@gestionmagnum.com.ar.
